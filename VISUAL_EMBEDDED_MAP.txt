🔧 TECHNICAL SPECS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

JavaScript Functions:
  • initializeEmbeddedMap()  - Initialize map & marker
  • updateLocation(lat, lng) - Update coordinates
  • reverseGeocode(lat, lng) - Get address from coordinates

Event Listeners:
  • marker.dragend           - When user stops dragging marker
  • map.click                - When user clicks on map

Alpine.js Data:
  • location.latitude        - Coordinate latitude
  • location.longitude       - Coordinate longitude
  • customer.address         - Address from reverse geocoding
  • customer.catatan_alamat  - Additional notes
  • mapLoading               - Loading state

APIs Used:
  • Google Maps JavaScript API - Map rendering
  • Google Geocoding API       - Reverse geocoding
