```bash
# 5. Push ke GitHub
git push -u origin main
```

### Option 2: Menggunakan Personal Access Token

Jika diminta password, gunakan Personal Access Token:

```bash
# Update remote dengan token
git remote set-url origin https://TOKEN@github.com/keyshanp/pempekbunda75.git

# Push
git push -u origin main
```

Ganti `TOKEN` dengan Personal Access Token dari GitHub.

## 🔑 Cara Buat Personal Access Token:

1. Login ke GitHub: https://github.com/login
2. Klik foto profil → **Settings**
3. Scroll ke bawah → **Developer settings**
4. **Personal access tokens** → **Tokens (classic)**
5. **Generate new token** → **Generate new token (classic)**
6. Isi:
   - **Note**: `Pempek Bunda 75`
   - **Expiration**: `90 days` atau `No expiration`
   - **Select scopes**: Centang **repo** (semua)
7. **Generate token**
8. **COPY TOKEN** (hanya muncul sekali!)

Direct link: https://github.com/settings/tokens/new
